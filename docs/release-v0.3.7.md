# Release v0.3.7

发布日期：2026-07-02

## 更新内容

- 总览页训练中心能量点重定位到楼顶目标位置，增强中心外发光，并扩展为 5 层扩散波纹。
- Manila-E 通过 `AI Operations Site` 进入时，对话调度保持默认收起，仅保留右侧竖向入口。
- Manila-E 底部员工按钮条改为贴近页面底部；当底部 dock 展开时自动上移，并与 dock 保持 20px 间距。
- Manila-E 剧情场景新增机器人参与者头像层，不同按钮按逻辑点亮告警、诊断、工单、排程、安全、运营、问数等智能体。
- 优化调度面板顶部状态提示，降低持续闪烁感。
- 强化告警点和告警任务卡：新增红色扩散环、扫描光效、呼吸高亮和更明显的连线。
- 强化 Outcomes Flow：增加上行数据流光数量与亮度，并在总览弹窗中加入步骤逐项完成动画和数字从 0 增长效果。
- 更新缓存版本参数到 `v0.3.7`，便于 GitHub Pages 刷新资源。
